# 🚗 ABG Motors Market Entry Analysis

> A Data Science Capstone Project that analyzes customer purchasing behavior in Japan and predicts potential buyers in the Indian market using Logistic Regression and Tableau.

---

## 📌 Project Overview

ABG Motors, a successful Japanese automobile manufacturer, plans to expand into the Indian market. This project analyzes customer data from Japan, builds a machine learning classification model, and predicts potential customers in India.

The project also includes an interactive Tableau dashboard to visualize customer demographics and purchasing trends.

---

## 🎯 Business Objective

- Analyze Japanese customer data.
- Build a Logistic Regression classification model.
- Predict potential customers in India.
- Help ABG Motors determine whether entering the Indian market is profitable.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
- Tableau
- Git & GitHub

---

## 📊 Dataset Features

- Current Age
- Gender
- Annual Income
- Age of Car

**Target Variable:**
- Purchase (0 = No, 1 = Yes)

---

## 🤖 Machine Learning Model

**Algorithm Used:** Logistic Regression

### Data Preprocessing

- Converted Maintenance Date to DateTime
- Created AGE_CAR feature
- Encoded Gender
- Standardized features using StandardScaler
- Train-Test Split (80:20)

---

## 📈 Model Performance

| Metric | Value |
|--------|------:|
| Accuracy | **68%** |
| Predicted Buyers | **67,231** |

---

## ✅ Business Conclusion

The model predicted **67,231** potential buyers in the Indian market, which is significantly higher than the required target of **12,000**.

Therefore, **ABG Motors has strong potential for successful market entry into India.**
